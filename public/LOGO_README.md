# Logo Placeholder

## Instructions

Please replace the placeholder logo files with your actual clinic logo:

1. **logo.png** - Main logo (recommended size: 200x60px or similar aspect ratio)
2. **favicon.ico** - Browser favicon (16x16px, 32x32px, 48x48px)

### Logo Requirements

- **Format**: PNG with transparent background
- **Color**: Gold (#D4AF37) on transparent background
- **Text**: "EG Elegant Glow Aesthetic Clinic"
- **Style**: Elegant, professional, luxury feel

### How to Replace

1. Save your logo as `logo.png` in this directory
2. Generate favicon from your logo using a tool like:
   - https://favicon.io/
   - https://realfavicongenerator.net/
3. Replace `favicon.ico` in this directory

### Temporary Placeholder

Until you add your actual logo, the website will display a text-based logo using the clinic name.
