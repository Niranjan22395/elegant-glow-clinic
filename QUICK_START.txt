   → Go to: https://github.com/new

Step 3: Upload Your Code to GitHub
   → Open PowerShell in this folder

🌐 ADD CUSTOM DOMAIN (OPTIONAL)
═══════════════════════════════════════════════════════════════════════════

Step 1: Buy Domain from GoDaddy
   → Go to: https://www.godaddy.com
   → Search: elegantglowclinic.com (or any name you want)
   → Purchase domain (~₹500-1000/year)

Step 2: Add Domain to Render.com
   → In Render dashboard, go to your site
   → Click "Settings" → "Custom Domains"
   → Click "Add Custom Domain"
   → Enter your domain: elegantglowclinic.com
   → Render will show you DNS records

Step 3: Update DNS in GoDaddy
   → Login to GoDaddy
   → Go to "My Products" → Click your domain
   → Click "DNS" → "Manage Zones"
   → Add these records (Render will provide exact values):
   
   Type: A
   Name: @
   Value: [Render will provide IP]
   
   Type: CNAME
   Name: www
   Value: [your-site.onrender.com]

Step 4: Wait & Verify
   → Wait 15-30 minutes for DNS to update
   → Visit your domain
   → DONE! Your site is live with custom domain! 🎉
